## Salesforce Object Model

### Account
Represents customer organizations.
Types:
- School
- Business
- Organization

### Contact
Represents individuals associated with accounts.
Roles:
- Buyer
- PTO Member
- Administrator

### Opportunity (Apparel Order)
Represents a single apparel order.
Key Fields:
- Garment Type
- Quantity
- Decoration Method
- Rush Order

Stages:
Inquiry → Quoted → Approved → In Production → Completed → Closed Lost
