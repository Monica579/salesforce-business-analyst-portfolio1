# Salesforce Business Analyst Portfolio
## Case Study: South Side Girl Apparel

This repository showcases an end-to-end Salesforce Business Analyst portfolio
using a real custom apparel business as the example client.

## What This Portfolio Includes
- Business Requirements Documentation (BRD)
- Salesforce data model and object design
- User stories and Jira-ready backlog
- Salesforce Admin build steps
- Automation with Flows and validation rules
- End-user training documentation
- Live Salesforce demo script for interviews

This portfolio demonstrates how I translate business needs into scalable
Salesforce solutions while supporting delivery, adoption, and reporting.
